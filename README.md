<h1>북북클럽</h1>
 <ul>
     <li>'책에 대해서만 이야기하는 SNS가 있었으면 좋겠다'에서 시작한 프로젝트</li>
   <li>X(구. 트위터)의 책 버전이라고 생각하면 되겠다!</li>
 </ul>
<br>
<h2>관심사 및 목표</h2>
 <ul>
    <li>김영한님의 Spring boot + JPA 강의에서 배운 내용들을 적용해보자.</li>
   <li>프로젝트의 완전한 완성보다 배운 것을 써먹어보는 뜯어먹는 프로젝트의 용도로 사용해보자.</li>
 </ul>
 <br>
 <h2>구현 기능</h2>
 <ul>
   <li>로그인/회원가입</li>
   <li>회원정보 조회/수정</li>
   <li>타임라인 등록/수정</li>
 </ul>
<br>
<h2>사용 기술</h2>
 <ul>
    <li>JAVA, Spring boot, JPA, HTML, Javascript, Bootstrap</li>
 </ul>
